# 2DROP

Drops two topmost items off the top of the stack

Input stack: `a b`

Output stack: 

## Allocation

None

## Errors

[EmptyStack](./ERRORS/EmptyStack.md) error if there are less than two items available on the stack

## Examples

```
0x10 0x20 2DROP =>
```
